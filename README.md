# RGUKT AI Assistant

## Project Title
RGUKT AI Assistant

## Features
- Intelligent assistance for students.
- Provides study materials based on coursework and queries.
- Chatbot integration for real-time assistance.
- Resource recommendation engine.

## Tech Stack
- **Frontend:** JavaScript, React
- **Backend:** Python, Flask
- **Database:** PostgreSQL
- **Deployment:** Docker, AWS

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/depavathnareshnaik/rgukt-ai-assistant.git
   ```
2. Navigate to the project directory:
   ```bash
   cd rgukt-ai-assistant
   ```
3. Install the dependencies:
   ```bash
   npm install
   # For backend
   pip install -r requirements.txt
   ```
4. Set up the database and environment variables as needed.
5. Run the application:
   ```bash
   npm start
   # For backend
   python app.py
   ```
